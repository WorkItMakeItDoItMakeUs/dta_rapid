DTA Rapid Prototyping Theme

This Jekyll theme uses CSS and javascript written by the Digital Transformation Agency to make it easy to quickly prototype online user journeys with a government-y look and feel.

Content can be written as HTML or markdown (see: daring fireball - markdown). For fancier components and reuse we use Jekyll includes (see: the Jekyll documentation on includes).
